How to Build?
-------------

To initialize your local repository using the HaxogenOs trees, use a 
command like this:

```bash
  repo init -u git://github.com/haxogenOS/Manifest.git -b hax-n
  
  Then to sync up:
----------------

```bash
  repo sync
```
Finally to build:
-----------------

```bash
  . build/envsetup.sh
  brunch <device_name>
```
